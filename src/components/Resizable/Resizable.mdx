# Resizable

## Description

**Resizable** enables drag-to-resize panels; use it for split views and dashboards where users control layout proportions.

## Anatomy

- **ResizableHandle**
- **ResizablePanel**
- **ResizablePanelGroup**

## Usage

```tsx
import { ResizablePanelGroup, ResizablePanel, ResizableHandle } from "@devbrock/comic-ui";

export function Example() {
  return (
    <ResizablePanelGroup direction="horizontal" className="h-40 w-[520px] rounded border">
      <ResizablePanel defaultSize={50}>
        <div className="p-4 text-sm">Left</div>
      </ResizablePanel>
      <ResizableHandle />
      <ResizablePanel defaultSize={50}>
        <div className="p-4 text-sm">Right</div>
      </ResizablePanel>
    </ResizablePanelGroup>
  );
}
```
