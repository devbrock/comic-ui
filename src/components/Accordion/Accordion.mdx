# Accordion

## Description

**Accordion** is a reusable UI primitive, migrated from the original shadcn-style implementation.

## Anatomy

## Usage

```tsx
import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from "@devbrock/comic-ui";

export function Example() {
  return (
    <Accordion type="single" collapsible>
      <AccordionItem value="item-1">
        <AccordionTrigger>What is Comic UI?</AccordionTrigger>
        <AccordionContent>A punchy, comic-inspired component set.</AccordionContent>
      </AccordionItem>
    </Accordion>
  );
}
```
